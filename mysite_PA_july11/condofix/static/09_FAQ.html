Les profils dans CondoFix¶

  • C’est quoi les profils dans CondoFix ==> *CondoFix fut élaboré autour de quatre (4) profils. Le « Login », les fonctionnalités permises ou pas, le menu d’aide, etc… ont été élaborés autour de ces quatre (4) profils. Les profils sont Copropriétaire, Concierge / Employé, Membre du C.A. et Administrateur Condofix. Une même personne pourrait avoir plus d’un profil (exemple: un membre du C.A. qui est aussi copropriétaire). La meilleure façon d’en apprendre davantage sur les profils est d’aller au début du menu d’aide et d’accéder aux différents profils.

  • Que puis-je faire avec mon profil Copropriétaire? ==> Cliquez `ici <03_Profil_Coproprietaire.html>`__

  • Que puis-je faire avec mon profil Concierge / Employé? ==> Cliquez `ici <04_Profil_Concierge_Employe.html>`__

  • Que puis-je faire avec mon profil Membre du C.A.? ==> Cliquez `ici <05_Profil_Membre_du_CA.html>`__

  • Que puis-je faire avec mon profil Administrateur CondoFix? ==> Cliquez `ici <06_Profil_Administrateur_CondoFix.html>`__
